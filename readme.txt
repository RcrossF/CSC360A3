Final assignment for Uvic's CSC 360 - Operating Systems. 

Welcome to diskman! Diskman gives you control over a FAT12 filesystem.

How to use:
1. Compile by running "make" in terminal
2. Run according to usage below

Usage:

./diskinfo {target disk}

./disklist {target disk}

./diskget {target disk} {target file}

./diskput {target disk} {target file}
